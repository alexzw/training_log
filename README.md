# 🏋️ Training Log

> 一個簡潔的個人健身追蹤網站，Apple 風格設計，支援多用戶，所有資料儲存在本地瀏覽器。

🌐 **線上網址：** `https://alexzw.github.io/training-log/`

---

## ✨ 功能

| 功能 | 說明 |
|------|------|
| 👥 多用戶 | 支援多人使用，每人資料完全獨立 |
| 📊 總覽儀表板 | 體重趨勢圖、本月訓練次數、連續訓練天數 |
| ✏️ 今日記錄 | 記錄體重、訓練類型、時間、動作內容與備注 |
| 📋 訓練歷史 | 所有記錄列表，可按類型篩選 |
| ⚖️ 體重追蹤 | 折線圖視覺化體重變化，標示目標體重 |
| 📷 身材照片 | 上傳並瀏覽歷史身材照片 |
| 🗓 訓練計劃 | 建立各階段訓練計劃，顯示進度條 |
| 🎯 目標追蹤 | 設定目標體重與每週訓練次數，圓形進度視覺化 |

---

## 🚀 使用方式

直接開啟網頁即可，無需安裝任何東西。

1. 點擊側邊欄頭像區域 → **新增用戶**，輸入你的名字
2. 從左側導覽切換各功能頁面
3. 每日在「今日記錄」填寫訓練內容
4. 在「目標追蹤」設定你的目標體重與訓練頻率

> **資料儲存於瀏覽器 localStorage，清除瀏覽器快取會遺失資料，建議固定使用同一瀏覽器。**
